# Java Calculator

A simple menu-driven calculator application built in Java. This project performs common mathematical operations and provides an easy way to practice Java fundamentals such as loops, conditionals, user input, and the Math library.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation (Power)
- Division-by-zero handling
- Continuous calculations until the user chooses to exit

## Technologies Used

- Java
- Scanner Class
- Switch Statements
- Loops
- Java Math Library

## Project Structure

```text
CalculatorProject/
│
├── Calculator.java
└── README.md
```

## How to Run

### Compile

```bash
javac Calculator.java
```

### Run

```bash
java Calculator
```

## Example Usage

```text
=== Calculator ===
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus
6. Power
7. Exit

Choose an option: 3

Enter first number: 10
Enter second number: 5

Result = 50.0
```
